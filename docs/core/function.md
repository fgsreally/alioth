# 编辑器功能模块
:::info
目前，`alioth`的功能模块重度依赖于`phecda`系列的库（一个状态管理库），

详见[phecda-vue](./phecda-vue.md)

:::


在预设中，我们需要知道应用此时此刻的状态，也需要方式去更改它，

使用过`codemirror`等编辑器的不会陌生--你总是需要知道此时选中的内容，或想把选中内容高亮，功能模块就是干这个的
